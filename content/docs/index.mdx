## What is Systaliko UI?

**Systaliko UI is a collection of React components** built with [Tailwind CSS](https://tailwindcss.com/) and [Motion](https://motion.dev/), Inspired by [shadcn/ui](https://ui.shadcn.com/), it is components distribution not "npm install" where you install only the components you want not the whole library. Making the components fully customizable and flexible to different usecases, where you can easily adapt the components to your own design and brand, and also you can create a different variant of the same component.
