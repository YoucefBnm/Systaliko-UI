## Usage

```tsx
<motion.div
  key={animation}
  variants={animationVariants}
  transition={{ duration: 0.3, delay: 0.2, ease: 'easeInOut' }}
  initial="hidden"
  whileInView="visible"
  className="text-4xl tracking-tight font-bold"
>
  <h2>Animation variants demo</h2>
</motion.div>
```
