## Usage

```tsx
<ShaderGradient
  colors={["#8C59D9", "#D973A6", "#73A6F2"]}
  animate
  intensity={1.2}
  density={0.8}
  className="absolute inset-0"
/>

// Hurricane — full control
<ShaderHurricane
  colors={["#00f2fe", "#4facfe", "#ffd700"]}
  background="#0a0a0a"
  intensity={1.5}
  density={0.6}
  speed={1.2}
  className="absolute inset-0"
/>
```
