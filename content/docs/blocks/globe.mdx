## Usage

```tsx
<Globe>
  <GlobeGlow className="bg-background/50" />
  <GlobeSvg>
    <GlobePin x={280} y={100} delay={0} />
    {/* Europe */}
    <GlobePin x={640} y={80} delay={1} />
    {/* Asia */}
    <GlobePin x={900} y={120} delay={2} />
  </GlobeSvg>
</Globe>
```
