## Usage

```tsx
<ImagePlayer
  images={IMAGES}
  interval={200}
  renderImage={(src) => (
    <Image
      src={src}
      width={400}
      height={300}
      className="size-full h-auto max-h-full max-w-xl object-cover inline-block align-middle"
      alt="showcalse"
    />
  )}
/>
```
