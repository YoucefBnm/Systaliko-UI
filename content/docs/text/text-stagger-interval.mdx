## Usage

```tsx
<TextStaggerInterval
  words={['Modern UI', 'Creative Layouts', 'Dynamic Sites', 'Sleek Designs']}
  animation={'blur'}
  staggerValue={0.05}
  interval={3000}
  pauseOnHover={false}
/>
```
